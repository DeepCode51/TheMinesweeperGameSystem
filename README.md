## 功能介绍（完整见运行截图）
玩家：界面上方有计时器和雷数显示，帮助掌握游戏进度。通过点击方格进行操作，未翻开的方格可能隐藏地雷，翻开后若无雷则显示周围雷数。右键点击可标记疑似雷的位置，避免误触。目标是排除所有非雷方格并成功标记所有雷的位置。
